# Centro de Control Orion

Dashboard en tiempo real para la misión Artemis II, utilizando datos oficiales de la API JPL Horizons de NASA.

## Créditos
- **API Client**: Ryan Fox (artemis-ii-cli, MIT License)
- **Dashboard**: elpocitano@gmail.com
- **Datos**: NASA JPL Horizons (nave -1024)

## Características
- Telemetría real de Orion (posición, velocidad, distancias)
- Clima solar (NOAA SWPC)
- Feed de noticias de Artemis II
- Fases de misión con visualización tipo reloj
- Trayectoria SVG interactiva
- Actualización automática cada 30 segundos
