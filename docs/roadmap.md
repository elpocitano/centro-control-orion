# Roadmap del Proyecto: Centro de Control Orion

## Propósito
Construir un dashboard educativo sobre la misión Artemis II y la cápsula Orion, con foco en visualización clara, rigor científico y arquitectura modular.

## Estado Actual
- **Fase 0 (Completada):** Prototipo funcional en un único archivo HTML.
- **Sprint 01 (En curso):** Refactorización estructural y limpieza de estilos.

## Próximos Sprints
- **Sprint 02:** Modularización de JavaScript (Configuración y Dominio).
- **Sprint 03:** Extracción de lógica de datos (APIs NASA, NOAA, DONKI).
- **Sprint 04:** Reorganización de UI por componentes (Cards, Mapas, Gráficos).
- **Sprint 05:** Accesibilidad, Mobile First y optimización de carga.
